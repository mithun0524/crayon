# Crayon Web Application

**Live Site:** [https://crayon-umber.vercel.app/](https://crayon-umber.vercel.app/)

This is the frontend landing page and web application for Crayon. 

## Tech Stack
- Next.js 16 (App Router)
- React 19
- Tailwind CSS v4
- Turbopack

## Development

```bash
pnpm install
pnpm dev
```
